# SmartLibrary

Учебный C++ проект библиотеки с консольным интерфейсом.


## Структура проекта

```text
books/    - модели книг
library/  - основная бизнес-логика библиотеки
search/   - поиск по каталогу
users/    - модель пользователя
main.cpp  - консольный UI
```

## Требования

- C++17
- CMake 3.16+
- GCC / Clang / MSVC с поддержкой C++17

## Сборка

```bash
cmake -S . -B build
cmake --build build
```

## Запуск

```bash
./build/SmartLibrary
```

На Windows:

```powershell
.\build\Debug\SmartLibrary.exe
```

## Как пользоваться

После запуска откроется консольное меню:

1. показать доступные книги
2. показать пользователей
3. показать активные выдачи
4. добавить печатную книгу
5. добавить пользователя
6. выдать книгу
7. вернуть книгу
8. выполнить поиск

Проект стартует с предзаполненными тестовыми данными.

## Что можно улучшить дальше

- сохранить данные в JSON или SQLite
- добавить `EBook` и `AudioBook`
- сделать авторизацию без хранения пароля в открытом виде
- покрыть бизнес-логику тестами
- вынести UI из `main.cpp` в отдельный модуль
